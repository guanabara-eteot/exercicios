Formatação de Textos (Parte 1)

Formatação básica

Podemos dizer que um texto é forte (tag STRONG) e ele vai ficar escrito em negrito. Também existe a maneira de usar negrito sem ter tanto significado semântico usando a tag B.

Também podemos dar ênfase em um texto e ele vai ficar escrito em itálico usando a tag EM. De maneira similar podemos usar a tag I para deixar um texto itálico sem ter um significado semântico.

Outros tipos de formatação

Se você achar interessante, pode marcar um texto com um destaque com a tag MARK para que o visitante possa lê-lo com grande atenção. É como se usássemos caneta marca-texto em uma folha de papel.

Sabe aquelas letras pequenas de contrato e que ninguém lê?. Pois podemos usá-los em nossos sites usando a tag SMALL.

Outra formatação que pode ser usada é riscar um texto que não deve ser considerado, mas que ainda assim deve aparecer. Fazemos isso usando a tag DEL.

Ao contrário do texto deletado (riscado) mostrado acima, podemos destacar um texto com sublinhado usando a tag INS.

Também podemos usar textos sobrescritos como em xa+b usando a tag SUP ou subscritos como em H2O usando a tag SUB.
