Formatação de Textos (Parte 2)

Vamos continuar a aprender alguns tipos de formatação de textos.

Trechos de código

Se precisarmos escrever trechos de código em nosso site, como um código-fonte em JavaScript, podemos usar a tag CODE para isso. Se juntarmos a ela a tag PRE, o texto já vem previamente formatado.

            
    // Código JS para calcular o dobro de um número
    let n = Number(window.prompt("Digite um número:"))
    let d = n * 2
    window.alert(`O dobro de ${n} é igual a ${d}`)

Citações e abreviações

Outro recurso muito importante para sites são as citações. As citações curtas podem usar a tag Q e as longas usam a tag BLOCKQUOTE. Vamos ver exemplos:

Como diria minha avó: Quando não tiver o que dizer, não diga nada!

Como disse Steve Jobs:

Nascemos, vivemos por um momento breve e morremos. Tem sido assim há muito tempo. A tecnologia não está mudando muito este cenário.

Se formos usar siglas, podemos usar a tag ABBR para dar significado a elas. Por exemplo, passe o mouse sobre as siglas HTML e CSS e aguarde alguns segundos.
